# 開発仕様書

## アプリ名

不動産オーナー財務みえる化アプリ

## 基本コンセプト

決算書の大きな数字から、不動産オーナーが次の3点を確認できるようにする。

1. 利益が出ているか
2. 実際にお金が残っているか
3. 借入が重すぎないか

そのうえで、必要に応じて物件別CFへ分解する。

## 設計方針

### オーナー向けに難しくしすぎない

画面上の表現は専門用語を避ける。

| 内部計算 | 画面表示 |
|---|---|
| 概算キャッシュフロー | お金の残り方 |
| 賃貸収益倍率 | 家賃収入に対して借入が多すぎないか |
| 営業利益対借入率 | 借入に見合う利益が出ているか |
| CF完済年数 | 今の手残りで何年あれば返せるか |
| 修繕積立後CF | 将来修繕に備えてもお金が残るか |

## 画面一覧

### 1. 決算書の数字を入れる

入力項目。

- 年間家賃収入
- 駐車場収入
- その他収入
- 管理費
- 修繕費
- 固定資産税・都市計画税
- 保険料
- その他経費
- 支払利息
- 減価償却費
- 法人税等・所得税等
- 現在の借入金残高
- 年間元金返済額
- 未払金・長期未払金の年間返済額
- 将来修繕の年間積立目安
- 当年の現預金
- 前年の現預金
- 当期利益が分かる場合

### 2. お金の残り方を見る

出力。

- 年間収入
- 決算書上の利益
- 概算キャッシュフロー
- 修繕積立後の手残り
- 診断コメント

### 3. 借入の安全性を見る

3つの質問で表示する。

1. 家賃収入に対して、借入は重すぎないか
2. 借入に見合う利益が出ているか
3. 今の手残りで、何年あれば借入を返せるか

借入目標は次の3区分で表示する。

- 守りの目安
- 標準の目安
- 攻めの上限

### 4. 物件ごとに見る

物件A〜Eの匿名入力。

- 年間収入
- 年間経費
- 支払利息
- 減価償却費
- 年間元金返済
- 修繕積立目安

物件別に、利益、概算CF、修繕積立後の手残りを表示する。

### 5. 結果を印刷する

A4一枚レポート。

- 年間収入
- 決算書上の利益
- 概算CF
- 修繕積立後CF
- お金の残り方コメント
- 借入安全性コメント
- 次に確認したい資料

### 6. 次に見る資料

数字で違和感が出たときに確認する資料。

- 家賃明細、入居一覧、募集条件
- 修繕の請求書、見積書、修繕履歴
- 固定資産税納税通知書
- 火災保険証券、地震保険証券
- 借入返済予定表、金銭消費貸借契約書
- 役員貸付金、事業主貸、設備投資資料

## 判定ロジック

### お金の残り方

- 利益黒字、概算CF黒字、修繕積立後CF黒字：安定
- 利益黒字、概算CF赤字：黒字でも資金注意
- 概算CF黒字、修繕積立後CF赤字：修繕への備えに注意
- 利益赤字、概算CF黒字：利益と資金にズレ
- その他：守り優先

### 借入安全性

借入上限を3基準の最小値で計算する。

- 収益基準：年間賃貸収益 × 5
- 利益基準：営業利益 ÷ 10%
- CF基準：返済原資CF × 15

判定。

- 現在借入 <= 攻めの上限 × 70%：余裕あり
- 現在借入 <= 攻めの上限 × 85%：標準
- 現在借入 <= 攻めの上限：注意
- 現在借入 > 攻めの上限：守り優先

## 重要な注意点

このアプリは経営判断の入口であり、次の用途には使わない。

- 税務申告書の作成
- 金融機関の融資審査資料
- 不動産鑑定評価
- 投資勧誘
- 法的判断
